## How to run
1. Create a Python environment (recommended: conda) with packages used in the notebook (numpy, pandas, scikit-learn, matplotlib, seaborn, torch/keras for deep learning, nibabel for MRI, etc.).

```bash
python -m venv venv
source venv/bin/activate  # or `venv\Scripts\activate` on Windows
pip install -r requirements.txt
jupyter lab  # or jupyter notebook
```
